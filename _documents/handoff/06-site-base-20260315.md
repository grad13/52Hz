## 概要

Hub（orbital-q.sakura.ne.jp）の `.face` ディレクトリ規約を改善しました。news の分離、日付フォーマットの変更、サムネイル指定方法の追加が含まれます。

## 変更内容

### 1. news を `news.json` に分離

- `app.json` の `news` フィールドは廃止
- ニュースは `.face/news.json` に独立ファイルとして格納
- `news.json` が存在しない場合はニュースなし扱い（エラーにならない）

### 2. date を Unix timestamp に変更

- before: `{"date": "2026-03-10", "text": "..."}`
- after: `{"date": 1773100800, "text": "..."}`
- timestamp は UTC 00:00:00 基準

### 3. `thumbnail` フィールド追加（任意）

- `app.json` に `"thumbnail": "ファイル名"` を追加可能
- 指定するとファイル名を直接参照（glob 探索をスキップ）
- 未指定時は従来通り `thumbnail.*` を自動検出（後方互換）

## 依頼・申し送り

各プロジェクトで以下の移行を実施してください:

1. `app.json` の `news` 配列を `.face/news.json` に移動し、`date` を Unix timestamp に変換
   - 変換方法: `php -r "echo strtotime('YYYY-MM-DD UTC');"`
2. `app.json` から `news` フィールドを削除
3. サムネイルがある場合、`app.json` に `"thumbnail": "ファイル名"` を追加（任意だが推奨）
4. 変更を git commit & deploy

**急ぎではありません。** 現在は `news.json` が存在しない場合はニュースなし扱いになるため、移行前でもエラーにはなりませんが、旧フォーマットの `app.json` 内 news は Hub に表示されなくなります。
